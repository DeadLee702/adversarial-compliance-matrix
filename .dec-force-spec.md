## 3. SAFETY MANDATE (Hard Constraints)

### SIMULATOR MODE ONLY

✅ **ALLOWED:**
- File reading and analysis
- JSON report generation
- Logging and telemetry
- System introspection (read-only)

❌ **FORBIDDEN:**
- `os.remove()` — No file deletion
- `shutil.rmtree()` — No directory deletion
- `ctypes` — No kernel access
- Direct hardware memory manipulation
- Any destructive operations

### Output Requirements

**Every protocol must generate:**
- `[protocolname]_report.json`
- Contains field: `"warning": "Neniu dosiero estis modifita"` (No files were modified)
- ISO 8601 timestamps
- EsperantoProtokolo formatted logs

### Example Report Structure

```json
{
  "protocol_name": "Pendulastika Oracle",
  "protocol_number": 1,
  "timestamp": "2026-07-04T12:00:00Z",
  "status": "OPERATIONAL",
  "verdict": "PURA",
  "warning": "Neniu dosiero estis modifita",
  "findings": [],
  "closure": "Relenthol engaĝita."
}
```
